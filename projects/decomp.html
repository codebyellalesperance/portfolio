Machine Learning & Computing Projects

A collection of advanced computing projects showcasing deep technical expertise in low-level programming, performance optimization, and machine learning algorithms.

Project Overview

These projects represent my journey into the deeper aspects of computer science, from understanding how computers work at the lowest levels to implementing sophisticated machine learning algorithms. Each project was completed through pair programming, emphasizing collaboration and code quality.

The projects span from low-level systems programming in C and Assembly to high-level machine learning applications in Python. They demonstrate not just technical proficiency, but the ability to work effectively in teams and tackle complex, multi-faceted problems.

Universal Machine Emulator ⭐

Built a complete emulator for the Universal Machine (UM) architecture in C, implementing a virtual machine that could execute UM bytecode. This project required deep understanding of computer architecture, memory management, and instruction set design.

  • Architecture Implementation: Implemented 32-bit word architecture with segmented memory management
  • Instruction Set: Handled 13 different instruction types including conditional branching, memory operations, and arithmetic
  • Performance Optimization: Achieved 10x speedup through careful memory layout and instruction optimization
  • Error Handling: Robust error detection and reporting for invalid instructions and memory access violations

The emulator successfully executed complex UM programs and served as a foundation for understanding how virtual machines work in modern computing systems.

Advanced Image Compression

Developed a sophisticated image compression system using discrete cosine transform (DCT) and Huffman coding, achieving compression ratios comparable to JPEG while maintaining high image quality.

  • DCT Implementation: Implemented 2D discrete cosine transform for frequency domain analysis
  • Quantization: Designed adaptive quantization tables optimized for different image types
  • Huffman Coding: Built dynamic Huffman encoder for optimal entropy coding
  • Quality Control: Implemented adjustable quality settings balancing compression ratio and image fidelity

The compression system achieved 85% file size reduction on typical photographs while maintaining visually lossless quality, demonstrating deep understanding of signal processing and information theory.

Image Transformation & Cache Optimization

Created a high-performance image transformation library with sophisticated caching mechanisms, achieving significant speedups through careful memory management and algorithmic optimization.

  • Transformation Pipeline: Implemented rotation, scaling, filtering, and color space conversion operations
  • Cache System: Built multi-level cache hierarchy with LRU eviction and memory-aware sizing
  • Parallel Processing: Utilized SIMD instructions and multi-threading for performance-critical operations
  • Memory Optimization: Implemented zero-copy operations and memory pooling to minimize allocations

The library achieved 5x performance improvement over naive implementations through careful attention to memory access patterns and cache-friendly data structures.

RPN Calculator in Assembly

Implemented a complete Reverse Polish Notation (RPN) calculator in x86-64 Assembly language, demonstrating mastery of low-level programming and system interfaces.

  • Assembly Programming: Hand-coded all operations in x86-64 assembly with careful register management
  • Stack Operations: Implemented robust stack-based arithmetic with overflow/underflow detection
  • I/O Handling: Built custom input parsing and output formatting routines
  • Error Handling: Comprehensive error detection and user-friendly error messages

This project provided invaluable experience with low-level programming concepts and system programming that underlies all high-level languages and frameworks.
